## 1. サービス概要（※最後に書いてOK）

- このサービスはどんなサービスか（3行以内）
運動することで植物やキャラクターが成長する、ゲーミフィケーション型の運動習慣化アプリです。
ユーザーの運動ログに応じて育成状態が変化し、個人だけでなくチームでも成長を共有できます。
コミュニティ投稿機能により、運動を継続するモチベーションを高めます。

- 誰の、どんな課題を、どう解決するか
運動不足を自覚しているものの習慣化できない人の課題を、運動をゲームのように楽しめる仕組みで解決します。
運動することでキャラクターや植物が育つ達成感と、チーム機能による仲間との連帯感により、継続的な運動習慣を作ることを目指します。

## 2. このアイデアはどこから生まれたか

### 2-1. きっかけとなった体験・感情

このアイデアの元になった
**自分自身の体験・感情・違和感**を教えてください。

- いつ
エンジニア学習を進めている期間

- どんな状況で
長時間パソコンに向かって作業を続けていると、気づけばほとんど体を動かしていない日が続いていました。

- どんな気持ちになったか
運動不足であることは自覚しているものの、わざわざ運動するモチベーションが生まれず、
「運動を習慣にする仕組みがあれば続けられるのではないか」と感じました。

---

### 2-2. なぜそれが「気になった」のか

同じ体験をしても、気にしない人もいます。
**なぜ自分はそれが気になったのだと思いますか？**

- 自分の性格
習慣化が苦手で、モチベーションだけでは行動を継続できないと感じているため。

- 過去の経験
ゲームのように達成感があるものは継続しやすいと感じた経験があるため。

- 大切にしている価値観
日常の行動を少しでも楽しく継続できる仕組みがあれば、生活がより良くなると考えているため。

## 3. 課題の整理（表面的になっていないか）

### 3-1. 表に見えている困りごと

- 運動不足を自覚しているが、運動を習慣化できない
- 一人ではモチベーションが続かない
- 運動の成果が目に見えにくく、達成感を得にくい

---

### 3-2. 本当に解決したい課題は何か

- なぜ？
運動は成果が見えにくく、継続のモチベーションが保ちにくいから。

- なぜ？
一人で取り組む運動は楽しい体験になりにくいから。

- なぜ？
日常の中で運動をする明確な理由や楽しみがないから。

→ 本当に解決したい課題：
運動を「やらなければならないこと」ではなく、「ついやりたくなる体験」に変えること。

### 5-2. それでも自分が作りたい理由

- なぜ満足できなかったのか

他のユーザーと協力したり交流する仕組みが少ないと感じました。

- どこに違和感があったのか
運動習慣は一人では継続が難しい場合が多いにも関わらず、
個人利用が前提のアプリが多く、仲間と一緒に継続する仕組みが弱いと感じました。

## 7. このアプリで実現すること

### MVPで作る機能

- ユーザー登録 / ログイン

- 運動ログの投稿
  - 運動種目、運動時間、メモを記録できる。

- 植物 / キャラクターの成長機能
    - 運動ログを投稿するたびに「成長Pt」を付与。
    - 累計Ptが一定値を超えると、植物やキャラクターが数段階で変化する。
      - （例）種 → 芽 → 蕾 → 花
      - キャラクターの場合は装備や体の大きさなどが変化していく想定
    - 最終ログインから24時間が経過すると、キャラクターの場合は表情が「悲しい」、植物の場合は色が暗くなるなどに変化し、ユーザーへ継続を促す。
    - 最終ログインから24時間経過以降は、さらに24時間が経過するごとに、ポイントが段階的に減少していき、植物 / キャラクターの成長が後退していく。

- コミュニティ投稿機能
  - 1日の運動成果などを投稿して共有できる。
  - 投稿時に現在の自分のキャラクター / 植物の状態や成長ポイントが表示され、努力の過程を視覚的に共有できる。

- 運動履歴の表示
  - カレンダーUIで運動履歴を確認できる。
  - 運動した日は「成長した証」としてバッジアイコンが表示され、継続状況を一目で確認できる。


### 本リリースで作る機能

- チーム作成 / 招待機能
- チームでの共同育成機能
- コメント / いいね機能
- 運動ランキング

## 8. このアプリの懸念点とその対策

### 懸念点①

- 何が問題になりそうか：
ユーザーが運動ログを入力しなくなると、継続利用が難しくなる。

- なぜそう思うか：
習慣系アプリは継続が難しい傾向があるため。

- そのために考えている対策：
キャラクター成長やチーム機能など、運動を続けたくなる仕組みを取り入れる。
